# Cameron Brown — Portfolio

A personal portfolio website showcasing my projects and experience.

🔗 [cameronjoebrown.com](https://www.cameronjoebrown.com)

## Tech Stack
- **Framework:** Angular
- **Styling:** Bootstrap
- **Runtime:** Bun
- **Testing Framework:** Vitest
- **Deployment:** Vercel
- **DNS:** Amazon Route 53

## Running Locally
```bash
bun install
bun start
```

## Running Tests
```bash
ng test
```
